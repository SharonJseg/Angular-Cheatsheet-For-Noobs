Data binding - תקשורת בין האלמנטים

מבנה הקומפוננט

בדר"כ, כל עוד לא הגדרנו אחרת קומפוננט יכיל שלושה קבצים

  • app.component.ts - קובץ ה-Typescript (javascript)
  • app.component.html - קובץ ה-HTML
  • app.component.scss קובץ ה-SCSS או ה-CSS.

הקשר בין ה-TS ל-HTML Data binding

זהו הקשר בין ה-HTML לבין הנתונים והלוגיקה, הקישור של הנתונים בין מקובץ ה-TS לקובץ ה-HTML. לכל חלק ב-HTML יש קלאס המתאים לו ב-קובץ TS. וכל קטע בקובץ ה-HTML יכול להשתמש בשדות המוגדרים בקלאס (קומפוננט) שלו ורק בהם.

ישנם ארבעה סוגים עיקריים של קישור בין הלוגיקה לבין ה-html :

  1. String binding
  2. Property binding
  3. Event binding
  4. Two-way binding

1. String Binding (Interpolation) {{ interpolation }}

קישור נתונים של string יתבצע באמצעות {{ interpolation }} לדוגמא :

src/app/app.my-component.ts
export class myComponent{
       superHero: string = 'Sharon Segal'
}

המשתנה superHero יוצג בתוך ה- h1.

src/app/app.my-component.html
<h1>{{superHero}}</h1>

בתוך הסוגריים הכפולים ניתן להציג כל ערך שיכול לקבל המרה ל- string, ניתן להכניס חישובים מספריים לדוגמא:

src/app/app.my-component.html
<h1>1 + 1 = {{ calc }}</h1>

ואפילו פונקציה שמחזירה ערך, כיון שהערך המוחזר מקבל המרה ל-string .זה גם יכול להיות ערך בוליאני, וההמרה האוטומטית תהיה "false/true "לפי הערך. לדוגמא:

src/app/app.my-component.ts
export class myComponent{
       isFalse : boolean = false
       }
src/app/app.my-component.html
<h1>{{ isFalse }}</h1>

2. Property Binding {{ propertyBind }}

שינוי ערכי ה-properties של תגית HTML תיווצר על ידי סוגריים מרובעות, לדוגמא, ה-src של תגית התמונה תשתנה בהתאם לערך המקושר

src/app/app.my-component.html
<img [src]="itemImageUrl">
src/app/app.my-component.ts
export class myComponent{
     itemImageUrl : string = 'insert img url'
     }

3. Event Binding {{ eventBind }}

ניתן לקרוא לפונקציות בקובץ ה-TS עם Events שאותם צריך לעטוף בסוגריים עגולים לדוגמא :

src/app/app.my-component.html
<button {{clickBtn}}>Click Me</button>
src/app/app.my-component.ts
export class myComponent{
       isRight: boolean = false;

       enableBtn(){
       this.isRight=true;
       }
}
