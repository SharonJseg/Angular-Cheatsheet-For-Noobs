התקנה ראשונית של Angular

התקנת Angular CLI

בעזרת ה-CLI אנחנו יוצרים פרוייקטים, מקימים סיפריות חדשות, משימות פיתוח ופעולות נוספות. בכדי להתקין את ה- CLI:

npm install -g @angular/cli

יצירת פרוייקט וסביבת עבודה חדשה

נריץ את הפקודה ng new עם שם הפרוייקט:

ng new my-project-name

מייד לאחר מכן אנו נשאל אם אנחנו רוצים Angular routing, נבחר לא ואז נשאל אם אנו רוצים להשתמש ב-css/sass/scss.. בחרו באופציה שמתאימה לכם.

הפעלת האפליקציה לראשונה

וודאו שאתם נמצאים בסיפריה הנכונה, הסיפריה תהיה השם שנתתם לפרוייקט

cd my-project-name

האפליקציה מכילה שרת מקומי כמו nodemon כשעובדים על פרוייקט של Node.js והפקודה להריץ אותה היא:

ng serve --open

או בקצרה:

ng serve -o

ברירת המחדל של פורט השרת היא 4200, בכדי לשנות לפורט אחר:

ng serve -o --port=you-different-port-number

ייצוא פרוייקט

בכדי לייצא פרוייקט דרך ה- CLI צריך להקליד:

ng build my-project-name --prod

או בקצרה:

ng b my-project-name --prod
