קומפוננטות

מבנה הקומפוננט

כל קומפוננט מורכב משלושה חלקים עיקריים :

  • Selector - מציג את התגית הייחודית של הקומפוננט
  • TemplateURL \ Template - מציג את ה-HTML של הקומפוננט
  • StylesUrl \ Styles - מציג את הסטייל הייחודי של הקומפוננט

וכולם יהיו בתוך דקורטור בשם component@ לדוגמא :

@Component({
    selector: 'app-my-component',
    templateUrl: './my-component.component.html',
    stylesUrl: ['./my-component.css'],
    })

בנוסף יווצר בכל קומפוננט גם קובץ Specs

my-component.component.spec.ts

יצירת קומפוננט

בכדי ליצור קומפוננט יש להקליד:

ng generate component <component name> [options]

או בקצרה:

ng g c <component name> [options]

ניתן ליצור קומפוננט ללא שאר הקבצים. לדוגמא, ללא קובץ HTML :

ng g c <component name> -t

האופציות העיקריות :

האופציות תיאור
t- בכדי ליצור קומפוננט ללא קובץ HTML
s- בכדי ליצור קומפוננט ללא קובץ CSS / SCSS etc..
skipTests-- בכדי ליצור קומפוננט ללא קובץ SPECS
-t -s --skipTests ניתן לשלב את האופציות

Inline Elements בקומפוננט

במידה ולא יצרתם קובץ CSS או HTML, קובץ הקומפוננט ייראה כך : 

@Component({
          selector: 'app-my-component',
          template: '',
          styles: [],
          })

בכדי לכתוב קוד בתוך ה- inner template ועדיף לכתוב את הקוד בין backticks ` ` כדי לשבור שורות לדוגמא: 

@Component({
     selector: 'app-my-component',
     template: `<p>
                my html content
                </p> `,
     styles: [
              `p {
                color: cyan;
                }`
             ],
             })
